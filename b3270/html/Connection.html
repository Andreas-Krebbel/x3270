connection Indication

The host connection has changed state.

AttributeAlways present?Purpose
state yes New connection state
hostnoHost name, if connected

Connection states are listed here.

StateMeaning
not-connectedNo connection
resolvingResoving host name to address
pendingTCP connection pending
negotiatingTELNET option negotiation in progress
connected-initialTN3270 mode negotiation in progress
connected-nvtTN3270 NVT mode, line at a time
connected-nvt-charmodeTN3270 NVT mode, character at a time
connected-3270TN3270 3270 mode
connected-unboundTN3270E unbound mode
connected-e-nvtTN3270E NVT mode
connected-e-sscpTN3270E SSCP-LU mode
connected-tn3270eTN3270E 3270 mode

Example: 

<connection state="connected-tn3270e" host="foo.bar.com"/>
