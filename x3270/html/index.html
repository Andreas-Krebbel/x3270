What's x3270?
x3270 is an IBM 3270 terminal emulator for the X Window System.  (If you don't know what an IBM 3270 terminal emulator is, then you probably don't need x3270).  x3270 runs over a TELNET connection emulating either an IBM 3279 (color) or 3278 (monochrome).  It supports TN3270E, SSL/TLS, APL2 characters, non-English character sets including Russian, Turkish, Hebrew, and DBCS Chinese and Japanese, IND$FILE file transfer, NVT mode (emulating a color xterm), a pop-up keypad for 3270-specific keys, a scrollbar, and has extensive debugging and scripting facilities. It does not yet support graphics.  x3270 is distributed as source code, and can be used for free.

The Back-Level x3270 Version: 3.2
The current back-level release of x3270 is 3.2.20. Except for simple, show-stopping bugs, this release is no longer supported. It does not support SSL/TLS or DBCS.  You can download the source code from here.

c3270: Curses-Based 3270 Emulation
c3270 is the curses-based version of x3270.  It runs on any dumb terminal, and supports (almost) all of the features of x3270: 3279 color, TN3270E, IND$FILE file transfer, keymaps, alternate character sets, tracing, scripting, the ibm_hosts file, profile files ($HOME/.c3270pro), pr3287 printer sessions, etc.  c3270 scripts are compatible with x3270 scripts, and a subset of command-line options and resource definitions are compatible as well.

s3270: Pipe-Based Scripting
There is also a scripting-only version of x3270, s3270.  This program is intended primarily for writing "screen-scraping" applications, e.g., a CGI back-end script that retrieves database information from a mainframe.

tcl3270: Tcl-Based Scripting
There is also a Tcl-based 3270 scripting engine: tcl3270.  It lets you write Tcl scripts that manipulate 3270 sessions, and is quite a bit easier to set up and use than s3270.

pr3287: Printer Support
pr3287 is the printer companion for x3270, and allows printer output from a 3270 session to be directed to a Unix printer queue.

x026: 026 Keypunch Emulation

Other x3270 Resources
h3270 is a tool that allows IBM host access from within a web browser. It is based on s3270.
Carey Evans maintains x3270 for Debian.  He has a page that documents how to use x3270 to access AS/400 hosts.
Valerio Di Giampietro maintains an AS/400 and Linux page which uses s3270.
