This is the source tree for the 3270 emulation suite (x3270, c3270, s3270,
tcl3270, pr3287, wc3270, ws3270 and wpr3287).

General build instructions are in Webpage/Build.html.

If this source tree came directly from a repository, the configure scripts will
need to be generated first:
    make -f Makefile.aux prepare
